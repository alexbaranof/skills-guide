# Claude Skills Guide

A comprehensive review and collection of 800,000+ Claude Skills for builders, developers, and finance professionals.

## 📚 What's Included

- **skills-guide.html** — Interactive HTML guide with all skill repositories, recommendations, and installation instructions
- **Claude_Skills_Guide.pdf** — Full PDF guide for offline reference
- Three custom production-ready skills (see below)

## 🎯 Who This Is For

- **Construction Companies** — SLC, Oak Leaf, or similar
- **Web/App Developers** — React, Vue, Supabase, Next.js stack
- **Finance & Trading** — Portfolio tracking, invoicing, estimating

## 🚀 Quick Start

1. Visit **https://skills-guide.vercel.app** (or your custom domain)
2. Browse official repositories, marketplaces, and curated collections
3. Install custom skills directly into Claude:
   - modern-web-design.skill
   - london-materials.skill
   - london-garden-plan.skill

## 📦 Custom Skills

### modern-web-design.skill
Production-grade frontend architecture for React/Vue with:
- Three design systems (minimalist, bold brand-forward, dark fintech)
- shadcn/ui + Tailwind v4 patterns
- Accessibility (WCAG 2.1 AA), performance optimization
- SLC & Oak Leaf brand components

### london-materials.skill
Construction materials & supplier database:
- 8 categories (sanitary, tiles, kitchens, flooring, paints, lighting, joinery, building)
- Trade, Premium, Online, Specialist tiers
- London pricing (Q2 2026)
- Reverse-charge VAT support

### london-garden-plan.skill
Complete London garden design system:
- Planting plans (RHS-approved, London Clay-tolerant)
- Hardscape (paving, decking, pergolas)
- Drainage strategies
- Lighting, water features, outdoor kitchens
- Monthly maintenance schedules

## 🛠️ Installation

### In Claude Code
```bash
/plugin upload /path/to/modern-web-design.skill
/plugin upload /path/to/london-materials.skill
/plugin upload /path/to/london-garden-plan.skill
```

### In claude.ai
Settings → Capabilities → Skills → Upload skill → Select .skill file
